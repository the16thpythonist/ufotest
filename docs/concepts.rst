Concepts
========

A reference to documents which explain important concepts about the basic design and functionality of the
ufotest software which may at some points be required to understand the guides etc.

If a certain concept may be important to any other section, it is mentioned within that very section. So
there is no need to read all these in order, although the understanding may prove very beneficial.
